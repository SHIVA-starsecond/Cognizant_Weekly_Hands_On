## 📝 Week 1: Core Assignments

### 1️⃣ Software Design Patterns & Principles

Exploring foundational object-oriented design and SOLID principles to write scalable code.

* **Exercise 1: Singleton Pattern** ([View Source](https://www.google.com/search?q=Week1/Design_Patterns_and_Principles/Exercise1_SingletonPattern/SingletonPatternDemo.java))
* *Learning Outcomes:* Thread safety, double-checked locking mechanisms.


* **Exercise 2: Factory Method Pattern** ([View Source](https://www.google.com/search?q=Week1/Design_Patterns_and_Principles/Exercise2_FactoryMethodPattern/FactoryMethodPatternDemo.java))
* *Learning Outcomes:* Polymorphic object creation, Open/Closed Principle.



### 2️⃣ Data Structures & Algorithms (DSA)

Implementing core search algorithms and dynamic programming techniques.

* **Exercise 2: E-commerce Search Optimization** ([View Source](https://www.google.com/search?q=Week1/Algorithms_Data_Structure/Exercise2_EcommercePlatformSearch/EcommercePlatformSearch.java))
* *Learning Outcomes:* Linear Search `O(n)` vs. Binary Search `O(log n)`.


* **Exercise 7: Financial Forecasting Tool** ([View Source](https://www.google.com/search?q=Week1/Algorithms_Data_Structure/Exercise7_FinancialForecasting/FinancialForecasting.java))
* *Learning Outcomes:* Recursion, Memoization (Dynamic Programming), Big-O Analysis.



### 3️⃣ PL/SQL Fundamentals

Writing robust database scripts and stored procedures.

* **Exercise 1: SQL Control Structures** ([View Source](https://www.google.com/search?q=Week1/PLSQL_Exercises/Exercise1_ControlStructures/ControlStructures.sql))
* *Learning Outcomes:* `IF/ELSIF/ELSE`, `CASE` statements, and loop iterations (`WHILE`/`FOR`).


* **Exercise 3: Database Stored Procedures** ([View Source](https://www.google.com/search?q=Week1/PLSQL_Exercises/Exercise3_StoredProcedures/StoredProcedures.sql))
* *Learning Outcomes:* `IN/OUT` parameters, transaction management, exception handling.
